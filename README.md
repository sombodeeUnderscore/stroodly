# Stroodly

Stroodly is a compact AI agent builder. It shows how to create agents, attach tools, run them in different modes, and watch every step stream live.

## Quick tour

1. Open the home page.
2. Click **Open app**.
3. Try a few existing agents.
4. Build a new agent from a template or from scratch.
5. Open a run and review the steps.

## What to try

Good first tests:

- **Timer** — checks the time in a timezone
- **Page Taster** — summarizes a website
- **Head Baker** — does a multi-step task
- **Daily Bake** — finds information and formats it

## What to look for

Please focus on:

- Is it clear what the app does?
- Is it easy to find the next click?
- Do the labels make sense to a non-technical person?
- Does anything feel confusing, slow, or broken?
- Would you use it again?

## Main idea

The goal is to make the product easy to understand, easy to try, and easy to trust.
